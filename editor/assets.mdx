## Compatible File types

| Image | 3D | Audio | Video |
| :---  | :---  | :---  | :---  |
| `JPEG` `PNG` | `GLB` | `MP3` `WAV` | `MP4` |


## Max File and Texture Sizes
| | Free | Pro | 
| :---  | :---  | :---  | 
| 3D asset size | 10mb | 25mb |
| Audio file size | 3mb | 10mb |
| Video file size | 5mb | 15mb |
| Image texture size | 1024px | 2048px |
