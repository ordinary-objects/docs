---
title: "Viewport Navigation"
description: "How to move, rotate and pan the viewport"
---

<Note>
On Mac `cmd` is equivalent to `ctrl` on Windows
</Note>
### Trackpad

|  |                            |
| -------- | ------------------------------------- |
| Zoom in viewport direction     | cmd + two finger scroll                     |
| Zoom to specific object      | select object + cmd + two finger scroll                          |
| Rotate viewport   | left click + drag |
| Rotate around object   | select object + left mouse click + drag |
| Pan   | two finger scroll  |


### Apple Magic Mouse

|  |                            |
| -------- | ------------------------------------- |
| Zoom in viewport direction     | cmd + scroll                     |
| Zoom to specific object      | select object + cmd + scroll                          |
| Rotate viewport   | left click + drag |
| Rotate around object   | select object + left click + drag |
| Pan   | single finger scroll  |


### Standard Mouse

|  |                            |
| -------- | ------------------------------------- |
| Zoom in viewport direction     | cmd + scroll                     |
| Zoom to specific object      | select object + cmd + scroll                          |
| Rotate viewport   | left click + drag |
| Rotate around object   | select object + left click + drag |
| Pan   | wheel click + drag  |
