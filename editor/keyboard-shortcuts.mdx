---
title: "Keyboard Shortcuts"
description: "How to copy, paste and do other things with keyboard shortcuts"
---

<Note>
On Mac `cmd` is equivalent to `ctrl` on Windows
</Note>

|  |                            |
| -------- | ------------------------------------- |
| Copy selected hierarchy objects     | `cmd` / `ctrl` + `c`                      |
| Paste object(s)      | `cmd` / `ctrl` + `v`                          |
| Delete selected hierarchy objects   | `cmd` / `ctrl` + `backspace` |
| Refocus on a selected object (mouse in 3D view)   | `f` |
| Show scale gizmos (with object selected)   | `s` |
